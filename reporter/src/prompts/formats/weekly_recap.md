# Weekly Recap Format

## Purpose
Summarize the week's action: who won, who lost, standout performances, and notable transactions.

## Required Sections

### 1. Introduction (100-150 words)
- Hook with the week's biggest story
- Quick overview of the week's narrative
- Tease key matchups or storylines

### 2. Matchup Highlights (300-400 words)
Cover all games, emphasizing:
- Closest games (drama factor)
- Biggest blowouts (dominance factor)
- Upset alerts (expectation subversion)
- Key player performances in context

Format suggestion:
```
**Team A** def. **Team B**, 142.3-98.7
The narrative...
```

### 3. Standings Check (150-200 words)
- Current standings (top to bottom or grouped by tier)
- Playoff picture implications
- Who's rising, who's falling
- Magic numbers or elimination scenarios if applicable

### 4. Top Performers (100-150 words)
- Week's highest scorers (3-5 players)
- Context for their performance
- Fantasy implications

### 5. Transaction Report (100-150 words)
- Notable trades
- Key waiver pickups
- IR moves if significant
- Skip if no transactions

### 6. Looking Ahead (75-100 words)
- Key matchups next week
- Storylines to watch
- Playoff implications if late season

## Tone
- Energetic but informative
- Celebrate winners without diminishing losers excessively
- Balance analysis with entertainment

## Sample Opening

```markdown
# Week 8 Recap: Chaos Reigns in [League Name]

If you thought last week was wild, buckle up. Week 8 delivered four games decided by single digits, a 50-point blowout, and a waiver wire pickup that might just change the playoff picture.

Let's break it down.
```
